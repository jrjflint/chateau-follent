# Marketing Link References

Keep every QR code aligned with the Chateau Follent vintages archive by pairing each wine with its canonical anchor and UTM structure. These URLs help Lord James Follent trace how supporters discover the stories for each 2025 release.

## Standard QR UTM pattern
- `utm_source=qr` — QR scan attribution.
- `utm_medium=label` — distinguishes bottle labels from postcards or tasting menus.
- `utm_campaign=2025-<wine-slug>` — use the exact slug shown below so analytics match the correct entry.
- Append the section anchor after the query string: `...?utm_campaign=2025-<wine-slug>#wine-2025-<wine-slug>`.
